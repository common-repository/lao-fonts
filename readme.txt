== Description ==
Change Lao Font in a few simple steps

== Features ==
* Only Active a plugin and choose your favorite font, apply the change then you have nice Lao font on your Wordpress.
* Active ແລ້ວເຂົ້າໄປເມນູ Lao Fonts ທີ່ຢູ່ເມນູຂ້າງຊ້າຍ ແລ້ວເລືອກ Font ທີ່ຖືກໃຈ!


== Installation ==
1. Upload the "lao_fonts" files to 'wp-content/plugins/' directory.
2. Active the 'Lao Fonts' plugin.
3. Enjoy!
